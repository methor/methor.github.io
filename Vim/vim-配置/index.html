Vimdiff

E97

vimdiff ( 包括 git mergetool、fugutive 的 :Gdiff ) 都需要可执行文件 diff 的支持。在 MinGW 中如果出现 E97，可参考 解决 E97 中的做法，简而言之使用 ln -s /usr/bin/diff /usr/share/vim/vim74diff 将 vim74diff 指向系统 diff。

diff 输出内容的解读可以参考 how to read diff。
